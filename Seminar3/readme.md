# Wordle Game

A browser-based word guessing game with 6 attempts to find a 5-letter word. Built with HTML, CSS, and vanilla JavaScript.

## Features

- **6×5 grid** — Dynamically generated game board
- **Color feedback** — Green (correct position), yellow (wrong position), red (not in word)
- **Target word** — Currently hardcoded as "media"
- **Game logic** — Win/lose detection and attempt tracking

## Exercises

1. **Randomize word** — Create array of 5-letter words and select randomly
2. **Input validation** — Require exactly 5 letters before submission
3. **Center layout** — Use flexbox to center board and improve spacing
4. **New Game button** — Reset game state and select new word
5. **Keyboard support** — Allow Enter key to submit guess
6. **Show word on loss** — Display correct word in loss message
7. **Add animations** — Flip or fade effects when revealing colors
8. **Fix duplicate letters** — Handle repeated letters in guess/target correctly
9. **Visual polish** — Add title, improve colors, shadows, hover states, and statistics
