# Dynamic Chart (Canvas)

Real-time line chart using HTML5 Canvas and vanilla JS.

## Features

- **Canvas grid** — Clean grid and axis labels
- **Live data** — New point per second, shifting timeline
- **Styling** — Simple palette, centered layout

## Exercises

1. Add multiple series with distinct colors
2. Smoothing options (moving average or bezier)
3. Controls: start/pause, speed, range, grid toggle
4. Tooltips on hover with value/position
5. Switch chart types: line, bar, area, scatter
6. Stats panel: min, max, average, trend
7. Export PNG and add light/dark themes
